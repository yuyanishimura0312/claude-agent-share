Claude Code Agent Catalog

Claude Code のスラッシュコマンドとして使えるエージェントチーム。ワンコマンドでインストールして、すぐに使い始められます。

$ curl -sL https://yuyanishimura0312.github.io/claude-agent-share/install.sh | bash
はじめに

必要なもの

  • Claude Code がインストール済みであること
  • ターミナル（Mac / Linux / WSL）が使えること
  • curl コマンドが使えること（ほとんどの環境でデフォルトで利用可能）

インストール方法

1

全エージェントを一括インストール

ターミナルで以下を実行するだけです。~/.claude/commands/ にエージェントファイルが配置されます。

curl -sL https://yuyanishimura0312.github.io/claude-agent-share/install.sh | bash

2

個別にインストールしたい場合

特定のエージェントだけを選んでインストールできます。

curl -sL .../install.sh | bash -s -- deep-research cxo-meeting

3

Claude Code で使う

Claude Code を起動し、プロンプトに /deep-research AI倫理の最新動向 のようにスラッシュコマンドとテーマを入力するだけです。

使い方

基本的な使い方

A

スラッシュコマンドとして実行

Claude Code のプロンプトで /コマンド名 引数 と入力します。例えば、ディープリサーチなら /deep-research 再生可能エネルギー政策の国際比較 と入力してEnterを押すだけです。

B

エージェントが自動で動く

エージェントは指示に従って Web検索、論文検索、分析、レポート作成を自動で行います。途中で質問されることもありますが、基本的には待っているだけでOKです。

C

結果を受け取る

調査や分析が終わると、レポートやアクションプランが表示されます。追加の質問や深掘りもその場で依頼できます。

活用例

経営判断の検討

/cxo-meeting 新規事業として教育プログラムを開始すべきか
7人のCxOが財務・戦略・組織の観点から議論し、Go/No-Go判断を提示。

市場調査

/deep-research 日本のインパクト投資市場の現状と展望
Web+学術論文から体系的に調査し、出典付きレポートを作成。

競合分析

/competitor-scan Deloitte Monitor Institute
組織概要・サービス・ポジショニングを網羅的に調査。

アプリ品質チェック

/qa-review ~/projects/my-app
UI/機能/セキュリティを5人の専門家がレビューし、スコアリング。

ベストプラクティス調査

/best-practice リモートチームの心理的安全性
CMO+GRADEフレームワークで構造化された知見を提供。

学術翻訳

/translate-academic 社会的処方の概念と実践
人類学・社会科学の専門用語を正確に反映した英語に変換。

カスタマイズ

エージェントを自分用にカスタマイズする

1

ファイルを編集

~/.claude/commands/ にあるmdファイルを好きなエディタで開いて編集できます。プロンプトを変更したり、自分の業務に合った指示を追加してください。

2

新しいエージェントを作る

~/.claude/commands/my-agent.md を作成するだけで、/my-agent として使えます。$ARGUMENTS と書くとコマンドの引数が挿入されます。

3

更新する

新バージョンが出たら、既存ファイルを削除して再インストールしてください。rm ~/.claude/commands/deep-research.md → 再度インストールコマンド実行。

Claude Code Agent Catalog

Claude Code のスラッシュコマンドとして使えるエージェントチームのカタログ。ワンコマンドでインストールできます。

$ curl -sL https://yuyanishimura0312.github.io/claude-agent-share/install.sh | bash

インストール方法

  1. 上のコマンドをターミナルで実行すると全エージェントがインストールされます
  2. 個別インストール: curl -sL .../install.sh | bash -s -- deep-research
  3. 手動: エージェントファイルをダウンロードして ~/.claude/commands/ に配置
  4. Claude Code で /agent-name として使えるようになります
